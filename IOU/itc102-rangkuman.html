Perangkat Keras: CPU & Memori

Central Processing Unit (CPU)

CPU (prosesor) menafsirkan dan menjalankan instruksi dasar serta mengendalikan operasi komputer. Terdiri dari Control Unit (CU) - mengarahkan dan mengoordinasikan seluruh aktivitas komputer, serta Arithmetic Logic Unit (ALU) - melakukan operasi aritmatika (+, -, *, /) dan operasi logika (AND, OR, NOT, =, <, >).

Machine Cycle (Siklus Mesin)

4 tahap: Fetching (mengambil instruksi) → Decoding (menerjemahkan) → Executing (menjalankan) → Storing (menyimpan hasil).

Produsen prosesor PC terkemuka: Intel, AMD, VIA. Produsen prosesor perangkat mobile: Qualcomm (Snapdragon), Huawei (Kirin), Samsung (Exynos), Nvidia (Tegra).

Memori: RAM & ROM

Bit (binary digit) adalah unit data terkecil; Byte (256 karakter) adalah unit dasar memori.

JenisSifatContoh
Volatile MemoryKehilangan isi saat daya dimatikanRAM
Nonvolatile MemoryIsi tetap tersimpan meski daya dimatikanROM, flash memory, CMOS

Perawatan Komputer & Perangkat Mobile

Perawatan perangkat keras: bersihkan layar dengan kain lembap, bersihkan keyboard dengan compressed air, jaga ventilasi bebas debu. Perawatan perangkat lunak: hapus instalasi program tidak terpakai, defragment hard disk, instal software hanya dari sumber terpercaya. Menjaga baterai: isi penuh untuk perangkat baru, hindari cabut charger di tengah pengisian, nonaktifkan Wi-Fi/Bluetooth saat tidak dipakai, atur kecerahan layar.

Penyimpanan Data (Storage)

Storage vs Memory

Storage (nonvolatile) menyimpan data secara permanen untuk penggunaan di masa depan, sedangkan Memory (kebanyakan volatile) menyimpan data sementara. Storage device diklasifikasikan menjadi Sequential Access (magnetic tape) dan Direct Access (magnetic disk, optical disk, flash/memory card).

Jenis StorageContoh Penggunaan
Hard Disk (HDD)Sistem operasi, software, data pengguna - internal, eksternal, atau removable
Flash Memory/USBGambar digital, file untuk dibawa berpindah
CD/DVDSoftware, backup, film, musik
Cloud StorageSinkronisasi file, backup, kolaborasi dokumen (Google Drive, Dropbox, iCloud)

Head Crash & Pemulihan File

Penyebab head crash: jarak terlalu dekat antara head baca/tulis dan platter, debu/kotoran/rambut/asap, komponen usang, overheating, file rusak, lonjakan listrik. Tanda peringatan: komputer melambat, sering freeze, blue screen, suara aneh.

File yang dihapus sebenarnya tidak langsung hilang - hanya jalur aksesnya yang hilang. Cara pemulihan: jangan menyimpan file baru di disk tersebut, gunakan software freeware (Recuva, Recover My Files) atau jasa data recovery komersial.

Jenis Storage Lain & NAS/SAN

Magnetic stripe card, smart card (contact/contactless), microfilm/microfiche, RFID tag. NAS (Network Attached Storage) - server di jaringan dengan alamat IP sendiri. SAN (Storage Area Network) - jaringan kecepatan tinggi menghubungkan storage ke banyak server, bahkan lintas lokasi jauh.

Internet & World Wide Web

Evolusi Internet

Berakar dari proyek Pentagon's Advanced Research Projects Agency (ARPA) - beroperasi September 1969, menghubungkan 4 universitas: UCLA, UC Santa Barbara, Stanford Research Institute, University of Utah. Tujuan awal: menghubungkan ilmuwan lintas lokasi untuk proyek militer/sains, serta tetap berfungsi meski diserang. Tidak ada satu pihak pun yang memiliki Internet.

LAN (Local Area Network) - jaringan dalam satu organisasi. Intranet - bagian internet yang terbatas dalam satu organisasi. Protokol dasar: TCP/IP (Transmission Control Protocol/Internet Protocol).

IP Address & Domain Name

Setiap komputer yang terhubung internet punya alamat unik (IP address) - 4 kelompok angka 0-255 dipisah titik (contoh: 138.68.107.54). Domain Name System (DNS) menerjemahkan IP address ke teks yang mudah dibaca (contoh: www.iou.edu.gm). ccTLD (country code Top-Level Domain): sa (Arab Saudi), pk (Pakistan), in (India), id (Indonesia), my (Malaysia).

Komponen URL (Uniform Resource Locator): Protocol, Domain name, Path, Web page name.

Pencarian Efektif: Boolean Search

Istilah BooleanFungsiContoh
ANDMencari item yang memuat KEDUA kata kunci"adult learning AND online courses"
ORMencari item yang memuat SALAH SATU kata kunci"adult learners OR adult students"
NOTMENGECUALIKAN item dengan kata kunci tertentu"NOT bachelor programs"

Browsing Aman & Wi-Fi Publik

Panduan browsing aman: verifikasi keamanan situs, matikan location sharing, hapus riwayat browsing, jangan simpan kata sandi, gunakan phishing filter dan pop-up blocker, gunakan private browsing/proxy server. Panduan Wi-Fi publik: hindari mengetik kata sandi/info finansial, logout dari situs, matikan koneksi nirkabel saat tidak dipakai, waspada terhadap orang yang mengintip dari belakang bahu.

Keamanan Digital & Kejahatan Siber

Cybercrime & Jenis Serangan

Kategori pelaku kejahatan siber: hacker, cracker, script kiddie, corporate spy, unethical employee, cyberextortionist, cyberterrorist. Cyberwarfare - serangan menargetkan jaringan komputer pemerintah untuk melumpuhkan negara.

Jenis serangan internet/jaringan: Malware, Botnets, Denial of Service Attacks, Back Doors, Spoofing (e-mail spoofing, IP spoofing).

Cara Mengamankan Diri

Gunakan antivirus, waspada lampiran e-mail tak dikenal, pindai media removable sebelum dipakai, terapkan firewall, backup rutin. Untuk jaringan rumah: gunakan router/firewall, nonaktifkan macro security, perbarui software, tetap update informasi keamanan.

Autentikasi & Kontrol Akses

4 cara autentikasi identitas pengguna:

KategoriContoh
Something the individual knowsPassword, PIN, pertanyaan keamanan
Something the individual possessesSmartcard, kunci elektronik, kunci fisik
Something the individual is (static biometrics)Sidik jari, retina, wajah
Something the individual does (dynamic biometrics)Pola suara, tulisan tangan, ritme mengetik

CAPTCHA (Completely Automated Public Turing test) dikembangkan Carnegie Mellon University untuk membedakan manusia dan komputer.

Backup & Keamanan Jaringan Nirkabel

Backup: gunakan media berkualitas tinggi, simpan di tempat tahan api/panas, simpan salinan off-site. Mengamankan jaringan Wi-Fi rumah: ganti password default, ganti SSID (nama jaringan), jangan broadcast SSID, aktifkan enkripsi WPA2, aktifkan fitur MAC filtering, pilih lokasi router yang aman.

4 Kategori Definisi AI

Acting Humanly, Thinking Humanly, Thinking Rationally, Acting Rationally - AI mengandalkan algoritma untuk mencapai hasil, memadukan pencarian tujuan (goal seeking), pemrosesan data, dan akuisisi data.

Expert System (Sistem Pakar)

Muncul pertama tahun 1970-an, memakai pengetahuan pakar manusia yang disimpan dalam komputer untuk memecahkan masalah yang biasanya membutuhkan keahlian manusia. 4 komponen dasar: Knowledge Acquisition Facility (mengumpulkan keahlian dari berbagai sumber), Knowledge Base (menyimpan fakta dan aturan heuristik), Inference Engine (program yang menarik kesimpulan/solusi), Explanation Facility (membantu pengguna memahami bagaimana sistem mencapai keputusan).

3 cara representasi pengetahuan: Rule-based (pernyataan IF...THEN), Frame-based (basis data terorganisasi), Logic-based (teori himpunan untuk membangun relasi).

Aplikasi AI & Robotika

Contoh aplikasi AI: pengenalan wajah, game AI, pengenalan tulisan tangan, natural language processing, pengenalan suara, computer vision, Google Translate (memakai sistem Google Neural Machine Translation/GNMT). Kelas robot: manipulator, mobile, humanoid - dipakai dalam industri (Industry 4.0), bedah robotik (Da Vinci), layanan (Roomba, robot restoran), hingga eksplorasi lingkungan berbahaya (NASA Mars rover).

Proses Pemecahan Masalah & Logika Proposisi

4 fase pemecahan masalah: Goal formulation → Problem formulation → Search → Execution. Logika proposisi: pernyataan (proposisi) yang bernilai benar atau salah, dihubungkan dengan konektif (Conjunction/AND, Disjunction/OR, Negation/NOT, Conditional, Biconditional).

Ujian Tengah Semester - Modul 1-15 ▼
  • Apa itu komputer, dan apa hubungan antara data dan informasi?
    Jawaban: Komputer adalah perangkat elektronik yang memproses data menjadi informasi. Data adalah fakta mentah belum diproses (tidak bermakna sendiri); Informasi adalah data yang telah diproses menjadi bermakna bagi pengguna.
  • Bedakan tiga jenis era.
    Jawaban: Era Pertanian (pengolahan lahan/ternak), Era Industri (otomasi, revolusi industri), Era Informasi (fokus pada penciptaan, distribusi, dan pemakaian informasi - abad ke-21).
  • Sebutkan dan jelaskan lima komponen komputer.
    Jawaban: Hardware (perangkat keras), Software (perangkat lunak), Data, People (pengguna), dan Procedures (prosedur/langkah kerja) - kelima komponen ini bekerja bersama membentuk sistem informasi yang fungsional.
  • Apa itu control unit dan arithmetic logic unit pada prosesor?
    Jawaban: Control Unit (CU) mengarahkan dan mengoordinasikan seluruh aktivitas komputer serta mengatur pergerakan data antar memori/ALU/perangkat I/O. Arithmetic Logic Unit (ALU) melakukan operasi aritmatika (+, -, *, /) dan operasi logika/perbandingan (=, <, >, AND, OR, NOT).
  • Apa empat langkah dalam machine cycle?
    Jawaban: Fetching (mengambil instruksi), Decoding (menerjemahkan), Executing (menjalankan), Storing (menyimpan hasil).
  • Apa itu bit, dan bagaimana rangkaian bit merepresentasikan data?
    Jawaban: Bit (binary digit) adalah unit data terkecil bernilai 0 atau 1. Rangkaian 8 bit membentuk 1 byte, yang setara dengan satu karakter (256 kemungkinan kombinasi karakter).
  • Bagaimana cara kerja POS terminal dan ATM?
    Jawaban: Keduanya menggabungkan berbagai perangkat input dan output: POS terminal memakai keyboard, barcode scanner, card reader, dan printer struk untuk mencatat pembelian dan memproses pembayaran; ATM memakai magnetic card reader, keypad, touch screen, dan cash dispenser untuk transaksi perbankan.
  • Apa perangkat input alternatif untuk pengguna difabel?
    Jawaban: Termasuk dalam kategori assistive technology - seperti printer Braille (termasuk Braille Al-Qur'an, pertama diproduksi di Saudi Arabia 1985-1986), voice input, dan perangkat input yang disesuaikan kebutuhan khusus pengguna.
  • Apa empat jenis output?
    Jawaban: Text, Graphics, Audio, Video.
  • Apa perbedaan storage device dari storage media?
    Jawaban: Storage device adalah perangkat keras yang merekam dan mengambil data (mis. hard disk drive); storage media adalah materi fisik tempat data benar-benar disimpan (mis. platter di dalam hard disk, atau kepingan CD/DVD).
  • Apa itu cloud storage dan apa keunggulannya?
    Jawaban: Layanan penyimpanan data lewat internet (Google Drive, Dropbox, iCloud, dll). Keunggulan: sinkronisasi file lintas perangkat, backup otomatis, kolaborasi dokumen, dan akses dari mana saja.
  • Bagaimana kartu magnetic stripe dan smart card digunakan?
    Jawaban: Magnetic stripe card menyimpan data pada pita magnetik (kartu ATM/debit klasik). Smart card menyimpan data pada chip tertanam, tersedia dalam versi contact (harus disentuhkan) dan contactless (tap saja) - dipakai untuk pembayaran, identitas, dan akses.
  • Bedakan packaged software dari freeware.
    Jawaban: Packaged software dijual dalam bentuk paket komersial berlisensi (harus dibeli); freeware adalah software gratis untuk digunakan namun kode sumbernya tidak selalu terbuka/tidak boleh dimodifikasi bebas.
  • Jelaskan lima karakteristik aplikasi pengolah kata (word processing).
    Jawaban: Membuat dokumen (entri teks/gambar), mengedit (insert/delete/cut/copy/paste), memformat (font, ukuran, tampilan), fitur WYSIWYG ("What You See Is What You Get"), dan fitur tambahan seperti AutoCorrect, Grammar Checker, Mail Merge, dan Tracking Changes.
  • Apakah ada perbedaan antara aplikasi berbasis web dan berbasis cloud?
    Jawaban: Aplikasi web-based berjalan di server dan diakses lewat browser tanpa instalasi; aplikasi cloud-based secara spesifik menyimpan dan memproses data di infrastruktur cloud. Keduanya saling tumpang tindih namun cloud-based lebih menekankan penyimpanan/pemrosesan terdistribusi, bukan hanya cara akses.
  • Apa itu system software, dan apa dua jenis system software?
    Jawaban: System software adalah program yang memungkinkan aplikasi berinteraksi dengan komputer dan mengelola sumber daya internal/eksternal. Terbagi 4 kategori: Operating System, Utility Software, Device Driver, Language Translator (soal menyebut "dua jenis" - kemungkinan merujuk pembagian utama Operating System vs Utility Software).
  • Apa fungsi-fungsi sistem operasi?
    Jawaban: Memulai komputer, menyediakan antarmuka pengguna, mengelola program dan memori, mengoordinasikan jaringan, memantau kinerja, mengatur keamanan, menjadwalkan tugas dan mengonfigurasi perangkat, serta menyediakan manajemen file dan utilitas lain.
  • Apa tujuan berbagai utility programs?
    Jawaban: Melakukan tugas pemeliharaan sistem: Disk Cleanup (membersihkan file tak terpakai), Disk Defragmenter (merapikan susunan file di disk), Personal Firewall (memblokir akses tidak sah), Antivirus (mendeteksi malware), Backup/Restore Utilities (mencadangkan dan memulihkan data).
  • Apa keuntungan menggunakan jaringan?
    Jawaban: Berbagi sumber daya (printer, file, koneksi internet), memungkinkan komunikasi (e-mail, chat, video conference), memungkinkan kolaborasi real-time, dan sentralisasi manajemen data/keamanan.
  • Apa perbedaan LAN, MAN, dan WAN?
    Jawaban: LAN (Local Area Network) mencakup area terbatas seperti satu gedung/kampus. MAN (Metropolitan Area Network) mencakup area kota. WAN (Wide Area Network) mencakup area luas lintas kota/negara - internet adalah contoh WAN terbesar.
  • Bedakan perangkat komunikasi kabel (wired) dan nirkabel (wireless).
    Jawaban: Wired menggunakan kabel fisik (Ethernet, fiber optic) untuk transmisi data - umumnya lebih stabil dan cepat. Wireless menggunakan gelombang radio (Wi-Fi, Bluetooth) tanpa kabel fisik - lebih fleksibel namun rentan gangguan sinyal.
  • Jelaskan karakteristik khas bricks-and-mortar, click-and-order, dan clicks-and-mortar dalam e-Commerce.
    Jawaban: Bricks-and-mortar tidak memiliki kehadiran web sama sekali. Click-and-order menjual barang terutama lewat web tanpa toko fisik. Clicks-and-mortar memiliki kehadiran fisik maupun online sekaligus.
  • Jelaskan langkah-langkah dalam transaksi e-Commerce.
    Jawaban: (1) Pelanggan mengunjungi toko elektronik; (2) mengumpulkan barang di keranjang belanja; (3) memasukkan info pembayaran, dikirim ke bank; (4) bank melakukan pemeriksaan keamanan dan mengirim otorisasi; (5) server mengirim konfirmasi dan memproses ke pusat pemenuhan; (6) pusat pemenuhan mengemas dan mengirim laporan; (7) info pengiriman diposting; (8) pesanan diterima pelanggan.
  • Diskusikan kekhawatiran utama terkait pembayaran online.

  • Diskusikan evolusi web dengan menguraikan kontribusi/karakteristik khasnya.

    Jawaban: Cyberwarfare menargetkan jaringan komputer pemerintah untuk melumpuhkan suatu negara; keduanya (cyberwarfare dan cyberterrorism) sama-sama membutuhkan tim ahli, dana besar, dan perencanaan bertahun-tahun, namun cyberterrorism lebih menyasar teror/ketakutan publik sedangkan cyberwarfare menyasar kelumpuhan infrastruktur negara.

  • Apa itu cell address di Excel?
    Jawaban: Kombinasi huruf kolom dan angka baris yang menunjukkan lokasi unik sebuah sel di worksheet, misalnya A1 (kolom A, baris 1).
  • Apa yang dimaksud Relative cell referencing dan Absolute cell referencing di MS Excel?
    Jawaban: Relative reference (mis. A1) berubah otomatis mengikuti posisi baru saat formula disalin ke sel lain. Absolute reference (mis. $A$1) tetap merujuk sel yang sama persis meski formula disalin ke mana pun.
  • Bagaimana cara freeze panes di Excel?
    Jawaban: Pilih sel/baris/kolom acuan → tab View > Freeze Panes > pilih opsi (Freeze Panes, Freeze Top Row, atau Freeze First Column) agar baris/kolom judul tetap terlihat saat scroll.
  • Bagaimana cara membatasi seseorang menyalin sel dari worksheet Anda?
    Jawaban: Gunakan fitur Protect Worksheet (tab Review) untuk mengunci sel tertentu agar tidak bisa disalin/diedit tanpa kata sandi.
  • Apa perbedaan Formula dan Function di Excel?
    Jawaban: Formula adalah ekspresi perhitungan yang ditulis manual oleh pengguna (mis. =A1+A2); Function adalah formula bawaan/predefined siap pakai (mis. =SUM(A1:A4), =AVERAGE(), =COUNT()).
  • Sebutkan urutan operasi (order of operations) yang dipakai Excel saat mengevaluasi formula.
    Jawaban: Mengikuti kaidah matematika standar: Tanda kurung () → Pangkat/Eksponen → Perkalian dan Pembagian (kiri ke kanan) → Penjumlahan dan Pengurangan (kiri ke kanan).
  • Bagaimana formula untuk: kalikan nilai di sel A1 dengan 10, tambahkan hasilnya dengan 5, lalu bagi dengan 2?
    Jawaban: =((A1*10)+5)/2
  • Apa perbedaan COUNT, COUNTA, dan COUNTBLANK?
    Jawaban: COUNT menghitung sel berisi angka saja; COUNTA menghitung sel berisi apa saja (angka, teks, tanggal - tidak kosong); COUNTBLANK menghitung sel yang benar-benar kosong.
  • Apa pintasan untuk menambahkan filter ke tabel?
    Jawaban: Ctrl+Shift+L.
  • Bagaimana cara membuat hyperlink di Excel?
    Jawaban: Pilih sel → klik kanan > Link (atau Ctrl+K) → masukkan alamat URL/lokasi tujuan → klik OK.
  • Bagaimana cara menggabungkan beberapa string teks sel menjadi satu sel?
    Jawaban: Gunakan fungsi CONCAT atau CONCATENATE, misalnya =CONCAT(A1," ",B1), atau operator "&" seperti =A1&" "&B1.
  • Bagaimana cara memisahkan satu kolom menjadi 2 kolom atau lebih?
    Jawaban: Pilih kolom → tab Data > Text to Columns → pilih Delimited (dipisah karakter seperti koma/spasi) atau Fixed Width, ikuti wizard hingga selesai.
  • Bagaimana cara kerja fungsi IF() di Excel?
    Jawaban: =IF(kondisi, nilai_jika_benar, nilai_jika_salah) - menguji suatu kondisi logika dan menghasilkan salah satu dari dua nilai tergantung hasil ujinya.
  • Apa itu Pivot Table?
    Jawaban: Tabel interaktif untuk meringkas, mengelompokkan, dan menganalisis data dalam jumlah besar secara fleksibel, dilengkapi fitur slicer dan timeline untuk penyaringan visual.
  • Diberikan tabel siswa - tulis fungsi untuk menambahkan status lulus/gagal pada kolom hasil berdasarkan kriteria tertentu.
    Jawaban: Umumnya memakai fungsi IF, contoh: =IF(B2>=60,"Lulus","Gagal") - menyesuaikan sel referensi dan nilai batas kelulusan sesuai kriteria soal.
  • Animation Painter bekerja mirip Format Painter di Microsoft Office. Jelaskan cara kerjanya.
    Jawaban: Pilih objek yang sudah memiliki animasi → klik Animation Painter (tab Animations) → klik objek lain untuk menyalin animasi yang sama persis, tanpa perlu mengatur ulang dari awal.
  • Untuk membuat satu bagian slide memakai template desain berbeda dari slide lain, apa yang harus dilakukan?
    Jawaban: Gunakan fitur Section (klik kanan pada slide panel > Add Section) untuk mengelompokkan slide, lalu terapkan desain/tema berbeda khusus pada section tersebut lewat Design tab dengan opsi "Apply to Selected Slides".
  • Dengan custom animation, bagaimana membuat teks muncul huruf demi huruf di slide?
    Jawaban: Pilih teks → Animations > pilih efek entrance (mis. Appear/Fade) → buka Effect Options > atur Animate Text menjadi "By Letter" agar teks muncul satu huruf pada satu waktu.
  • Anda sudah mengustomisasi template desain di satu presentasi dan ingin memakainya di presentasi lain. Apa cara terbaik melakukannya?
    Jawaban: Simpan slide/tema tersebut sebagai file Template (.potx) lewat Save As, kemudian buka file template itu saat membuat presentasi baru, atau gunakan fitur "Reuse Slides" untuk mengimpor slide dari presentasi lain.
  • Saat presentasi berlangsung dan perlu kembali ke slide beberapa slide sebelumnya, bagaimana caranya?
    Jawaban: Klik kanan saat dalam Slide Show > pilih "See All Slides" untuk melihat thumbnail seluruh slide dan langsung memilih slide yang dituju, atau ketik nomor slide lalu tekan Enter.
  • Tombol apa yang ditekan untuk masuk Slide Show view dan selalu mulai dari slide pertama?
    Jawaban: Tombol F5.
  • Apa fungsi Notes pane di bagian bawah jendela PowerPoint?
    Jawaban: Tempat menulis catatan pembicara (speaker notes) yang tidak ditampilkan ke audiens saat presentasi, namun bisa dilihat presenter di Presenter View sebagai panduan saat berbicara.
  • Sebutkan beberapa cara membuat presentasi baru di PowerPoint.
    Jawaban: Blank Presentation dari awal, memakai template/tema bawaan, mencari template online, atau mengimpor/menggunakan kembali (Reuse) slide dari presentasi lain.
  • Dalam konteks animasi PowerPoint, apa itu trigger?
    Jawaban: Trigger adalah objek yang, ketika diklik, akan memicu (menjalankan) animasi pada objek lain - memungkinkan animasi berjalan sesuai interaksi presenter, bukan otomatis berurutan.
  • Bagaimana membuat halaman catatan pembicara yang menampilkan slide, catatan terkait, dan logo perusahaan di setiap halaman?
    Jawaban: Buka View > Notes Master untuk mengatur tata letak halaman catatan, tambahkan logo/elemen tetap di Notes Master tersebut sehingga muncul konsisten di semua halaman catatan saat dicetak.
  • Apa yang dimuat dalam lingkungan (environment) PowerPoint?
    Jawaban: Ribbon dengan tab-tab fungsi, Slide Panel (thumbnail slide), area edit slide utama, Notes pane, Status bar, dan Quick Access Toolbar.
  • Apa itu backstage view?
    Jawaban: Tampilan yang muncul saat mengklik menu File, memuat info dokumen, opsi simpan/cetak/bagikan, dan pengaturan aplikasi - terpisah dari area pengeditan utama.
  • Bagaimana cara mengustomisasi slide layout?
    Jawaban: Buka View > Slide Master > pilih/edit layout yang ingin diubah (posisi placeholder, font, warna) - perubahan akan berlaku ke semua slide yang memakai layout tersebut.
  • Jika seseorang tidak memiliki PowerPoint, bagaimana ia bisa melihat presentasi tersebut?
    Jawaban: Menggunakan PowerPoint Online/Web (gratis via akun Microsoft), mengekspor presentasi ke format PDF atau video, atau memakai aplikasi pemutar PowerPoint Viewer gratis.
  • Tipe data apa yang valid di Microsoft Office Access?
    Jawaban: Short Text, Long Text, Number, Date/Time, Currency, AutoNumber, Yes/No, OLE Object, Hyperlink, dan Attachment.
  • Apa tujuan validation rule pada field properties (dalam table design view)?
    Jawaban: Membatasi nilai yang boleh dimasukkan ke suatu field agar sesuai kriteria tertentu (mis. angka dalam rentang tertentu), mencegah kesalahan input data.
  • Tipe data apa yang sebaiknya dipilih untuk field kode pos (zip code)?
    Jawaban: Short Text (bukan Number) - karena kode pos bisa memiliki angka nol di depan dan tidak pernah dipakai untuk perhitungan matematis.
  • Jelaskan dua karakteristik penting Primary Key.
    Jawaban: (1) Nilainya harus unik untuk setiap record (tidak boleh ada duplikat); (2) Tidak boleh kosong (null) - setiap record wajib memiliki nilai primary key.
  • Jelaskan tiga jenis relasi yang dapat dibuat di Access, beserta contohnya.
    Jawaban: One-to-One (satu record tabel A berpasangan dengan tepat satu record tabel B, mis. karyawan dan nomor induk pegawainya), One-to-Many (satu record tabel A berpasangan dengan banyak record tabel B, mis. satu pelanggan dengan banyak pesanan), Many-to-Many (banyak record saling berpasangan lewat tabel penghubung, mis. siswa dan mata kuliah).
  • Identifikasi berbagai cara membuat tabel baru di Microsoft Office Access.
    Jawaban: Table Design View (mendefinisikan field manual), Datasheet View (langsung mengetik data), mengimpor data dari Excel/sumber lain, atau memakai Table Templates bawaan.
  • Jelaskan perbedaan antara akses pemirsa (viewer) dan peran kolaborator di Google Docs.
    Jawaban: Viewer hanya bisa melihat dokumen tanpa mengubah apa pun. Collaborator (Commenter atau Editor) bisa menambahkan komentar/saran (Commenter) atau langsung mengedit isi dokumen (Editor).
  • Jelaskan bagaimana pengguna dapat menyimpan respons yang diterima memakai Google Forms.
    Jawaban: Respons otomatis tersimpan di tab Responses pada form itu sendiri, dan bisa dihubungkan ke Google Spreadsheet baru atau spreadsheet yang sudah ada untuk diolah lebih lanjut.
  • Bedakan tools kolaborasi online dari tools document management system, sertakan contoh yang sesuai.
    Jawaban: Tools kolaborasi online fokus pada interaksi real-time (chat, video conference, whiteboard) - contoh: Microsoft Teams, Google Meet, GoToMeeting. Document management system fokus pada penyimpanan, akses, dan pelacakan perubahan dokumen - contoh: Google Docs, OpenKM, LogicalDOC.
  • Bedakan antara "acting humanly" dan "thinking humanly" dalam kategori Artificial Intelligence.
    Jawaban: "Acting humanly" berfokus pada sistem yang berperilaku seperti manusia dari luar (mis. lolos Turing Test) tanpa harus meniru proses berpikirnya; "Thinking humanly" berfokus pada sistem yang benar-benar meniru proses kognitif/cara berpikir manusia secara internal.
  • Jelaskan tiga cara pengetahuan dapat direpresentasikan dalam Expert System.
    Jawaban: Rule-based (pernyataan IF...THEN), Frame-based (basis data terorganisasi/frame), Logic-based (memakai teori himpunan untuk membangun relasi antar fakta).
  • Diskusikan empat fase proses pemecahan masalah.
    Jawaban: Goal formulation (merumuskan tujuan), Problem formulation (merumuskan masalah), Search (mencari solusi), Execution (menjalankan solusi).
  • Apa manfaat dan kekurangan interaksi media sosial?
    Jawaban: Manfaat: mempererat hubungan, mendorong sosialisasi, sarana dakwah dan penyebaran ajaran Islam. Kekurangan: minimnya kontak manusia langsung, keterbukaan berlebih atas kehidupan pribadi, dan penyalahgunaan kebebasan berbicara.
  • Jelaskan mengapa etika e-mail (e-mail etiquette) itu penting.
    Jawaban: Mencerminkan karakter pribadi, etos kerja, dan perhatian terhadap detail; menyampaikan profesionalisme dan efisiensi dalam komunikasi tertulis, terutama karena e-mail bisa diteruskan/disimpan sebagai catatan permanen.
  • Identifikasi berbagai risiko penggunaan ponsel dalam jangka panjang dan jelaskan cara pencegahannya.
    Jawaban: Risiko: Repetitive Strain Injury (RSI), Computer Vision Syndrome (CVS), kecanduan internet, gangguan postur. Pencegahan: istirahat rutin, batasi waktu layar, jaga jarak pandang layar, terapkan ergonomi saat memegang perangkat, dan sadari tanda-tanda kecanduan digital sejak dini.
